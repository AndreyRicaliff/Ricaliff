# IFPB — Estudos de Programação

Repositório pessoal de exercícios e anotações do curso.

## Estrutura

```
src/           # código principal e experimentos
exercicios/    # exercícios por disciplina
anotacoes/     # resumos e notas de aula
```

## Uso

```bash
pnpm install
pnpm dev        # roda src/index.ts
pnpm build      # compila para dist/
pnpm lint       # checa erros de estilo
pnpm test       # roda testes
```
