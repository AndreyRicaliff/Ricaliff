# P3 v2 — Paridade com o rice do OS (Persona-Quickshell)

**Data:** 2026-07-03
**Escopo:** elevar o restyle P3 v1 (2026-07-01) à paridade com a linguagem visual do rice Hyprland+Quickshell do CachyOS.
**Fonte da verdade:** `~/.config/quickshell/Persona-Quickshell` (QML) + `~/.config/hypr/hyprland.lua`.

## Método

Extração de **design tokens** do QML (cor, easing, duração, forma) em vez de porte de código. Todo alpha de acento vira `color-mix(in srgb, var(--primary) N%, transparent)` — pré-requisito do toggle de tema.

## Mapa OS → web

| Elemento do rice | Origem (QML) | Implementação web |

| Acento `#0099FF` / ring `#1CC9D9` | `Data/ThemeMode.qml` | `--primary` / `--ring` |
| Toggle Win+T persona↔pro (`#7c8494` + ícone neutro) | `theme-toggle.sh` + `ThemeMode.qml` | `:root[data-theme="pro"]` + botão 🎭 + tecla `T` + swap do `sb-mark-img` |

## Dashboard enxuto (pedido do Ricaliff, 2026-07-03)

Só os 4 widgets de gestão: **Foco do dia · Progresso · Hoje · Essa semana**. Removido `#focus-proj-section` (faixa FOCO + lista chapada) — duplicava o Foco do dia e ficou fora da estética.

## Acessibilidade

`prefers-reduced-motion` pula o wipe (via `matchMedia` no `go()`) e desliga shake/stagger. Tecla `T` ignora foco em `input/textarea/select/[contenteditable]`.
